# 压缩图片和视频

根据分辨率压缩图片和视频。使用 [ImageMagick](https://imagemagick.org/) 压缩图片（会自动安装），使用 [ffmpeg](https://ffmpeg.org/) 压缩视频（已自带）。

## 用法

`dotnet script compress-media.csx -- [path] [OPTIONS]`

参数：

- `path`：要处理的文件目录，默认为当前目录。

选项：

- `-r, --recursion`：是否递归处理子目录，默认为 `true`。
- `-i, --image`：图片压缩选项，默认为 `2160`，具体的选项值请参见下面。
- `-v, --video`：视频压缩选项，默认为 `1080r`，具体的选项值请参见下面。
- `-b, --bitrate`：视频码率限制，会压缩高于指定码率的视频，默认为 `10000000`。
- `-e, --ext`：压缩目标后缀，使用空格分隔多个后缀，会自动识别类型，默认为 `.jpg .mp4`。

支持的图片和视频压缩选项：

| 选项值   | 说明 |
|:-------:|------|
| `0`     | 不处理 |
| `720`   | 压缩到不超过 1280×720 |
| `720r`  | 压缩到不超过 1280×720 或 720×1280 |
| `1080`  | 压缩到不超过 1920×1080 |
| `1080r` | 压缩到不超过 1920×1080 或 1080×1920 |
| `1440`  | 压缩到不超过 2560×1440 |
| `1440r` | 压缩到不超过 2560×1440 或 1440×2560 |
| `2160`  | 压缩到不超过 3840×2160 |
| `2160r` | 压缩到不超过 3840×2160 或 2160×3840 |
